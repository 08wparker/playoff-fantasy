# Playoff Fantasy Football 

This is a web-based application for a small group of friends to play an NFL fantasy football game for the playoffs.

## Rules
* Standard point-per-reception (PPR) fantasy scoring
* Roster: 1 QB, 2 RB, 3 WR, 1 TE, 1 D/ST, 1 K
* You can choose any player you want, but can only use them *once* during the playoffs.

## Application structure
* basic user create and account and login
* tab 1: select weekly roster from available players (rostered NFL players on the playoff teams that weekend MINUS the players previously played by the user)
* tab 2: scoring sheet
